## KAVEdit

这是一款短视频编辑 SDK。目标是仿抖音 APP 音视频处理功能!

**计划:**

**v1.0.1**

> 正在研发中...

- 基于 FFmpeg 、OpenGL/SL 自研万能音视频播放器（已完成）。
- 音频视频录制合成 mp4（计划中...）。
- 音频，视频剪辑合成（计划中...）。
